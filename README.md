# 🔐 Password Strength Checker

## 📌 Overview

This project is a Python-based Password Strength Checker that evaluates the security level of a given password. It helps users understand whether their password is strong enough to resist common attacks.

## 🎯 Objective

To analyze passwords based on multiple criteria and classify them as:

* Weak ❌
* Moderate ⚠
* Strong ✅

## ⚙️ Working

The program checks:

* Length of the password
* Presence of uppercase letters (A-Z)
* Presence of lowercase letters (a-z)
* Numbers (0-9)
* Special characters (e.g., @, #, $)

Based on these conditions, the password is scored and categorized.

## 💡 Example

Input:

```
Password123!
```

Output:

```
Strong Password ✅
```

## ▶️ How to Run

```bash
python password_checker_p1.py
```
